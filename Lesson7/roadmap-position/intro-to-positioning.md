# Вступ до позиціонування

Позиціонування - це один з ключових аспектів CSS, який дозволяє контролювати розташування елементів на веб-сторінці. Розуміння різних методів позиціонування є фундаментальним для створення складних і гнучких макетів.

## Що таке позиціонування в CSS?

У CSS позиціонування визначає, як елементи розташовуються на сторінці відносно:
- Нормального потоку документа
- Елемента-контейнера
- Інших елементів на сторінці
- Вікна браузера (viewport)

Для контролю розташування елементів використовується властивість `position` і ряд пов'язаних властивостей: `top`, `right`, `bottom`, `left` та `z-index`.

## Базова схема роботи

```css
.element {
  position: type; /* статичне, відносне, абсолютне, фіксоване або липке */
  top: value;     /* зміщення зверху */
  right: value;   /* зміщення справа */
  bottom: value;  /* зміщення знизу */
  left: value;    /* зміщення зліва */
  z-index: value; /* контроль накладання */
}
```

## Типи позиціонування

CSS пропонує п'ять основних типів позиціонування:

1. **Static** - стандартне позиціонування, елементи розташовуються згідно з нормальним потоком документа.

2. **Relative** - елемент позиціонується відносно свого початкового положення в нормальному потоці.

3. **Absolute** - елемент видаляється з нормального потоку та позиціонується відносно найближчого попередника з не-статичним позиціонуванням.

4. **Fixed** - елемент видаляється з нормального потоку та позиціонується відносно вікна перегляду браузера.

5. **Sticky** - гібрид відносного та фіксованого позиціонування, елемент поводиться як відносно-позиціонований до досягнення певної позиції прокрутки.

## Чому позиціонування важливе?

Розуміння позиціонування дозволяє вам:

- Створювати складні макети з точним контролем
- Розміщувати елементи один над одним
- Фіксувати елементи на певних позиціях при прокрутці
- Розробляти адаптивні інтерфейси для різних розмірів екранів
- Реалізовувати сучасні UI-компоненти (спливаючі вікна, підказки, навігацію)

## Сумісність з браузерами

Більшість сучасних браузерів підтримують всі типи позиціонування. Проте при використанні `position: sticky` варто пам'ятати:

- Internet Explorer не підтримує sticky позиціонування
- Для кращої кросбраузерної підтримки рекомендується використовувати префікси:

```css
.sticky-element {
  position: -webkit-sticky; /* Для Safari */
  position: sticky;
}
```

## Наступні кроки

Ознайомившись з основними концепціями позиціонування, перейдемо до вивчення кожного типу окремо, починаючи зі [статичного позиціонування](./types/static-positioning.md).
