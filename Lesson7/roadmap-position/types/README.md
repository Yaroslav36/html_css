# Типи позиціонування в CSS

Це розділ присвячений детальному огляду кожного типу позиціонування в CSS. Кожен файл містить повне пояснення, приклади використання та рекомендації.

## Типи позиціонування

1. [**Static позиціонування**](./static-positioning.md) - стандартне за замовчуванням
2. [**Relative позиціонування**](./relative-positioning.md) - відносно початкового положення
3. [**Absolute позиціонування**](./absolute-positioning.md) - відносно позиціонованого предка
4. [**Fixed позиціонування**](./fixed-positioning.md) - відносно вікна браузера
5. [**Sticky позиціонування**](./sticky-positioning.md) - гібрид відносного та фіксованого

## Порівняльна таблиця

| Тип позиціонування | Залишається в потоці документа | Створює контекст позиціонування | Реагує на прокрутку | Типове використання                                   |

| Absolute           | ❌                             | ✅                              | ✅                  | Спливаючі підказки, елементи керування                |
| Fixed              | ❌                             | ✅                              | ❌                  | Фіксована навігація, модальні вікна                   |
| Sticky             | ✅                             | ✅                              | ✅                  | Заголовки секцій, що прилипають                       |

## Як обрати правильний тип позиціонування?

При виборі типу позиціонування, запитайте себе:

1. **Чи елемент повинен залишатися в потоці документа?**

   - Так: використовуйте `static`, `relative` або `sticky`
   - Ні: використовуйте `absolute` або `fixed`

2. **Чи елемент повинен реагувати на прокрутку сторінки?**

   - Так: використовуйте `static`, `relative` або `absolute`
   - Ні: використовуйте `fixed`
   - Частково: використовуйте `sticky`

3. **Чи елемент повинен створювати контекст позиціонування для дочірніх елементів?**
   - Так: використовуйте будь-який тип, окрім `static`
   - Ні: можна використовувати `static`
