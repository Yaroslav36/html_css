# Шпаргалка з CSS позиціонування

## Типи позиціонування

### 1. Static (статичне)
- **Значення за замовчуванням**
- Елемент знаходиться в нормальному потоці документа
- Властивості `top`, `right`, `bottom`, `left` та `z-index` не застосовуються
- **Приклад:** `position: static;`

### 2. Relative (відносне)
- Елемент розташовується відносно свого початкового положення
- Зберігає місце в потоці документа
- Створює новий контекст позиціонування для абсолютно позиціонованих нащадків
- **Приклад:** `position: relative; top: 20px; left: 30px;`

### 3. Absolute (абсолютне)
- Елемент виключається з нормального потоку документа
- Позиціонується відносно найближчого позиціонованого предка
- Якщо позиціонованого предка немає, позиціонується відносно початкового блоку (зазвичай `<html>`)
- **Приклад:** `position: absolute; top: 0; right: 0;`

### 4. Fixed (фіксоване)
- Елемент виключається з нормального потоку документа
- Позиціонується відносно вікна перегляду
- Залишається на місці під час прокрутки сторінки
- **Приклад:** `position: fixed; bottom: 20px; right: 20px;`

### 5. Sticky (липке)
- Гібрид відносного та фіксованого позиціонування
- Елемент поводиться як відносно позиціонований, доки не досягне порогу прокрутки
- Після досягнення порогу, поводиться як фіксовано позиціонований
- **Приклад:** `position: sticky; top: 0;`

## Ключові властивості

### Offset Properties (властивості зміщення)
- **top:** відстань від верхнього краю
- **right:** відстань від правого краю
- **bottom:** відстань від нижнього краю
- **left:** відстань від лівого краю
- **Приклад:** `top: 50px; left: 25%;`

### z-index
- Керує порядком накладання елементів на осі Z
- Працює тільки для позиціонованих елементів (будь-який, крім static)
- Більше значення = ближче до користувача
- **Приклад:** `z-index: 100;`

## Найпоширеніші випадки використання

| Тип позиціонування | Випадки використання |
|-------------------|---------------------|
| **Relative** | • Невеликі зміщення елементів<br>• Створення контексту для абсолютно позиціонованих елементів |
| **Absolute** | • Спливаючі підказки<br>• Накладання елементів<br>• Іконки всередині елементів<br>• Елементи, які повинні бути розташовані в точній позиції |
| **Fixed** | • Фіксовані шапки/навігаційні панелі<br>• Кнопки "вгору"<br>• Модальні вікна<br>• Банери та оголошення |
| **Sticky** | • Прилипаючі навігаційні панелі<br>• Заголовки таблиць<br>• Заголовки розділів |

## Найкращі практики

1. **Використовуйте відносне позиціонування обережно:**
   - Намагайтеся уникати збільшення зміщення, яке може призвести до перекриття інших елементів.

2. **Працюючи з абсолютним позиціонуванням:**
   - Завжди встановлюйте `position: relative` для батьківського елемента, щоб створити контекст позиціонування.
   - Віддавайте перевагу використанню або `top`+`left`, або `bottom`+`right` для кращої підтримки RTL мов.

3. **Для фіксованого позиціонування:**
   - Враховуйте мобільні пристрої та забезпечте достатньо місця для вмісту.
   - Пам'ятайте про обмежену підтримку для мобільних браузерів при використанні `position: fixed` з `vh`/`vw` одиницями.

4. **Для липкого позиціонування:**
   - Завжди перевіряйте кросбраузерну сумісність.
   - Використовуйте поліфіли для старіших браузерів.

5. **Загальні рекомендації:**
   - Надавайте перевагу Flexbox та Grid для розмітки замість абсолютного позиціонування.
   - Використовуйте `z-index` інкрементально (10, 20, 30...) для легшого управління.
   - Тестуйте на різних розмірах екрана, особливо для позиціонованих елементів.
   - Визначте стратегію групування для `z-index` (шапка: 100-199, модальні вікна: 900-999 тощо).

## Швидкі шаблони коду

### Центрування елемента з абсолютним позиціонуванням
```css
.centered-absolute {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}
```

### Фіксована навігаційна панель
```css
.fixed-navbar {
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  z-index: 100;
}

body {
  padding-top: 60px; /* Висота навігаційної панелі */
}
```

### Липкий заголовок секції
```css
.sticky-header {
  position: sticky;
  top: 0;
  background-color: white;
  z-index: 10;
}
```

### Спливаюча підказка (tooltip)
```css
.tooltip-container {
  position: relative;
}

.tooltip {
  position: absolute;
  bottom: 100%;
  left: 50%;
  transform: translateX(-50%);
  margin-bottom: 5px;
  visibility: hidden;
  opacity: 0;
  transition: opacity 0.3s;
}

.tooltip-container:hover .tooltip {
  visibility: visible;
  opacity: 1;
}
```
