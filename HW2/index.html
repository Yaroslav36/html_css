CSS Grid Layout

CSS Grid — це двовимірна система сітки для CSS. Вона дозволяє створювати складні макети сторінок за допомогою рядків та стовпців одночасно, забезпечуючи повний контроль над розташуванням елементів.

На відміну від Flexbox, який переважно орієнтований на роботу в одному напрямку (або рядок, або стовпчик), Grid працює у двох вимірах. Це дає можливість створювати гнучкі інтерфейси, які легко адаптуються до будь-яких розмірів екрана без використання складних розрахунків ширини.

Еволюція Grid

Ідея створення сітки виникла ще на початку 2010-х років. Перша чернетка специфікації була опублікована командою Microsoft у 2011 році. Протягом кількох років стандарт доопрацьовувався W3C, і в березні 2017 року більшість сучасних браузерів (Chrome, Firefox, Safari) одночасно впровадили підтримку Grid, що стало справжньою революцією у веб-верстці.

Ключові поняття

GRID CONTAINER

Батьківський елемент, на якому застосовано display: grid. Він визначає межі всієї системи сітки

GRID ITEM

Прямі нащадки контейнера. Вони автоматично стають частиною сітки та можуть займати одну або декілька комірок.

GRID LINES

Розділові лінії, що утворюють структуру сітки. Ви можете прив'язувати елементи до конкретних номерів ліній.

GRID GAP

Відстань між рядками та стовпцями (зараз використовується властивість gap).
